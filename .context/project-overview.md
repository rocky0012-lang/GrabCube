# Project Overview: Secure Kenyan Real Estate Marketplace



## 1. Core Vision
A highly secure, two-sided web and mobile real estate platform designed specifically for the Kenyan market. The platform connects Tenants/Buyers with verified Property Owners for listings including apartments, hostels, houses, plots, bungalows, hotel rooms, motels, and Airbnbs.


## 2. Core Problem & Mission
The primary mission is to eradicate real estate fraud in Kenya. The system must prevent scammers from:
- Creating fake accounts using stolen identities.
- Uploading stock internet photos (e.g., Unsplash, Pinterest, Airbnb) to list fake properties.
- Conning users out of deposits via upfront M-Pesa scams.


## 3. Target User Roles
- **Tenants/Buyers:** Search, filter, view verified properties, geolocate items, and make secure escrow bookings.
- **Property Owners:** Register, pass identity verification, list properties, upload land documents, and manage bookings.


## 4. Monetization Strategy (Future)
- One-time verification/activation fee for owners (to cover API costs).
- Premium listing promotions or percentage commissions on bookings.
